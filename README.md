# AWS Storage and Database Lab

Hands-on AWS project simulating real-world Cloud Support tasks involving storage, databases, and serverless services.

S3, RDS, and Lambda setup with security, configuration, and troubleshooting.

---

## Overview
This project demonstrates core AWS cloud support skills related to storage, database management, and serverless computing.

I configured secure storage using S3, deployed and tested a relational database with RDS, and created a serverless function using AWS Lambda.

---

## Architecture

- S3 → object storage and static website hosting  
- RDS → managed relational database  
- Lambda → serverless compute for automated tasks  

---

## What I Built
- Created and configured S3 buckets with policies and permissions
- Enabled static website hosting on S3
- Deployed an RDS MySQL database and tested connectivity
- Performed basic database operations (CRUD)
- Created and tested AWS Lambda function
- Configured permissions and triggers

---

## AWS Services Used
- S3
- RDS
- Lambda

---

## Key Cloud Support Skills Demonstrated
- Storage configuration and access control (S3)
- Database deployment and connectivity (RDS)
- Serverless function setup (Lambda)
- Troubleshooting database connections and permissions
